# Review & Rating Application

## Tech Stack

Frontend:

* React.js
* React Router DOM
* Axios

Backend:

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose

## Features

* Add Company
* Company Listing
* Search Company
* Company Details Page
* Add Review
* Review Listing
* Average Rating
* Sort Reviews by Rating

## Setup

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Environment Variables

Create a .env file inside backend:

PORT=5000

MONGO_URI=your_mongodb_connection_string
